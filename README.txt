Online Medicine Shop (Two pages)
--------------------------------
How to use:
1. Unzip and open index.html in your browser. That's the public shop page.
2. Open admin.html to add/delete products. Admin password is: 11223344
3. Products are stored in browser localStorage on this PC. When you add in admin page they will appear on index.html automatically.
4. Email sending is not yet enabled. Orders are saved locally in browser localStorage. Later you can integrate EmailJS or a server.

Files:
- index.html   (public shop)
- admin.html   (password-protected admin panel)
